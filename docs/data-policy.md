# Data Policy

## Data Scope
- Data is limited to participating subreddits only.
- No cross-subreddit aggregation is performed.

## Retention
- Intermediate data is stored only as long as necessary for aggregation.
- Stored references are periodically purged.

## Deletion Handling
- If content or users are deleted or removed on Reddit, corresponding references are removed during the next processing cycle.

## Redistribution
- Raw Reddit content is not redistributed outside Reddit.
- Outputs consist of aggregated information and links back to original discussions.
